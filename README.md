# 🏠 ИЖС-Проектировщик

Android-приложение для планирования индивидуального жилищного строительства с офлайн ИИ-агентом и параметрическим 3D-конструктором.

[![Kotlin](https://img.shields.io/badge/Kotlin-1.9.22-blue)](https://kotlinlang.org)

## ✨ Возможности

### 📐 Параметрический 3D-конструктор
- Ввод параметров участка и дома
- Автоматическая генерация планировок
- 3D-просмотр модели
- Экспорт в PDF, OBJ, GLB

### 🤖 Офлайн ИИ-консультант
- **3 мини-агента** в одном приложении:
  - 🔸 Планировщик — зонирование, инсоляция
  - 🧮 Калькулятор — объёмы материалов, смета
  - ✅ Нормоконтроль — проверка отступов, этажности
- 100% офлайн-работа
- Системные промпты по СП 11-101-95, СП 55.13330, СП 42.13330

### 🏗️ Функции
- Сохранение проектов локально
- Расчёт площадей и объёмов
- Подбор материалов
- Проверка соответствия СНиП
- Светлая/тёмная тема

## 🏗️ Архитектура

```
ru/company/izhs_planner/
├── ai/                     # ИИ-агент и LLM
│   └── agents/             # 3 мини-агента
├── data/                   # Слой данных
│   ├── local/              # Room + DataStore
│   └── repository/         # Репозитории
├── domain/                  # Доменный слой
│   └── model/              # Модели
├── export/                  # Экспорт PDF/OBJ
├── generator3d/             # Параметрика
├── viewer3d/                # 3D-просмотр
├── mobile_ads/               # Яндекс Реклама
├── premium/                  # Премиум (IAP)
└── ui/                      # Compose UI
    ├── screens/             # Экраны
    └── theme/               # Тема
```
